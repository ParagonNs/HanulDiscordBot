# 한울 디스코드 봇
한울 봇은 한울 세계관의 일부를 사용하는 경제와 관련된 미니게임을 운영하는 봇입니다.
이제부터 여러분들은 한울 은하에서 성공을 위해 야심차게 시작하는 사업가입니다.
여러분들이 활동하는 장소는 네르메스 제국이며, 여러분의 모든 행동들이 제국과 상호작용이 일어납니다.
